# Dealership Search

APK is located in the /app/debug folder

Download the clone zip file
unzip to a folder on your computer
open the project in android studio
have API28 downloaded and installed

MainActivity lets a user build a search query to query the SQLite database on the device
and returns a list of cars that are displayed on the results activity.

DatabaseHelper.java could be turned into a Web API endpoint with a little modification to 
accept a JSON string
